# 🕵️‍♂️ Prompt Decoder

Welcome to **Prompt Decoder**, an engaging game and tool that challenges users to guess the hidden prompt behind an AI's response!

By utilizing state-of-the-art semantic matching (via SentenceTransformers) and localized AI hint generation (via Ollama), Prompt Decoder offers a robust and educational framework for exploring prompting structures, keyword identification, and semantic similarity.

## 🚀 Features

- **Semantic Similarity Scoring**: Understand how close your guess is conceptually using `all-MiniLM-L6-v2`.
- **Hybrid Scoring Engine**: Earn points based on semantic similarity, exact keywords, and structure.
- **Progressive Difficulty**: Unlock new and harder questions as you score points.
- **AI-Powered Hints**: Get contextual hints generated locally without giving the exact answer away (using `qwen2.5:1.5b` or your chosen Ollama model)!
- **Beautiful UI**: An immersive and fully responsive React UI that is engaging and modern.

## 🛠️ Technology Stack

- **Frontend**: React, Vite, TypeScript
- **Backend**: Flask, Python, SentenceTransformers
- **Local AI Engine**: [Ollama](https://ollama.ai/)
- **Styling**: Modern CSS

## 📋 Prerequisites

Before you begin, ensure you have the following installed:
- [Node.js](https://nodejs.org/en/) & npm
- [Python 3.8+](https://www.python.org/)
- [Ollama](https://ollama.com/) (If using local AI features)

## 🏗️ Setup & Installation

### 1. Start the Local AI (Ollama)
Ensure Ollama is running internally and download the necessary hint model:
```bash
ollama run qwen2.5:1.5b
```

### 2. Backend Setup
Navigate to the backend folder, set up your Python environment, and start the Flask server:
```bash
cd backend
pip install -r requirements.txt  # Or install: pip install flask flask-cors openai sentence-transformers
python app.py
```
*The backend server will run natively on `http://localhost:5000`.*

### 3. Frontend Setup
Navigate to the frontend's UI folder, install dependencies, and start the dev server:
```bash
cd frontend/promptdecoder-ui
npm install
npm run dev
```
*The web UI will be accessible locally via Vite.*

## 🎮 How to Play
1. The app will present you with an "Expected Output" generated by a hidden prompt.
2. Enter your best guess for what the original prompt was!
3. The hybrid scoring engine will break down your guess and provide semantic, keyword, and structure score.
4. If you miss, use AI-generated hints to refine your guess.
5. Succeeding advances your progress and unlocks higher levels!
